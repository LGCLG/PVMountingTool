# 🛠️ 光伏支架设计工具

基于GB50009-2012规范的光伏支架结构设计工具，支持风荷载、雪荷载计算及型钢选型。

## 📥 安装

1. 克隆仓库：
   ```bash
   git clone https://github.com/您的用户名/PVMountingTool.git
   cd PVMountingTool
安装依赖：

bash
pip install -r requirements.txt
🚀 使用
运行主程序：

bash
python pv_calculator.py

参数化光伏支架荷载计算
